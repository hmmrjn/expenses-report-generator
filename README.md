# Run Locally

```sh
$ git clone https://github.com/hmmrjn/expenses-report-generator.git
$ cd expenses-report-generator
$ bundle install
$ rails db:migrate
$ rails server
# Press Ctrl+C to quit.
```
Open http://localhost:3000/
